延展阅读

⚔️ 保护你的 API​

对于原生和单页应用来说，你需要访问一个或多个 API endpoint 以获取和更新数据。
了解更多关于如何辨认用户是谁以及保障你的 API 安全。

🛵 变更主要登录方式​

Logto 默认使用用户名与密码作为主要登录方式。
根据业务，你也许想要将验证码登录或者社交登录提升到主要区域中。

🧑‍🔬 创建你的连接器​

Logto 有一些内置的、开箱即用的连接器。
在我们开发更多连接器时，你也可以轻松创建自己的连接器。

🧑‍🚀 管理用户​

我们知道你关心用户管理和活动，我们也是。
了解更多关于如何洞察你的用户并以图形化的方式观察 DAU、MAU 等数据。

🌐 本地化​

从一个区域的业务到全球化企业，提供最好的用户体验的想法不会改变。
你可以无阻碍地更改当前的语言包内容或者添加一个新的语言。

🚀 部署​

上线的时刻来临时，放轻松，你可以慢一点。
我们将帮助你完成这个令人激动的旅程。
