🔗 在你的应用中集成 Logto

我们提供了一些 SDK 使你能轻松将 Logto 集成到你的客户端中。如果列表中没有涵盖你期望的平台 / 框架，请提交一个 feature request 或者贡献一个新 SDK。

原生应用​

在原生环境中运行的应用程序。

  • iOS
  • Android

通常原生应用需要与 API（resource）交流。参见 保护你的 API 以在 API 中鉴权。

单页应用​

在浏览器中运行并动态更新数据的应用程序。

  • 原生 JS
  • React
  • Vue

通常单页应用需要与 API（resource）交流。参见 保护你的 API 以在 API 中鉴权。

传统网页应用​

仅由 Web 服务器渲染和更新的应用程序。例如服务端渲染。

  • 传统 web
  • Next.js
  • Express
  • Go

Machine to Machine​

直接与 resource 通信的应用（通常没有 UI）。

  • Machine to Machine
